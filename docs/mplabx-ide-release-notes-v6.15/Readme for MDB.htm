1       Command Line Utility Defined

The Microchip Debugger (MDB) is a command-line interface to Microchip’s hardware and software development tools. As an alternative to using the Microchip MPLAB® X IDE (Integrated Development Environment) graphical interface, the MDB facilitates programming and debugging devices through a Command Prompt interface. The MDB is automatically installed with the MPLAB X IDE.

The MDB is designed for engineers who prefer to use the Command Prompt. The command-line interface to the debuggers is faster and allows more extensive testing to be performed. This is helpful for repetitive tasks or when automating a testing procedure.

The MDB provides a common tool for most Microchip debugging tools (see section 3) support for respective features based on the tool selected.

3       Development Tools Supported

The following development tools work with the MDB:

 

  • MPLAB ICD 4 In-Circuit Debugger
  • MPLAB ICD 3 In-Circuit Debugger
  • MPLAB PICkit™ 4 In-Circuit Debugger
  • PICkit 3 In-Circuit Debugger/Programmer
  • MPLAB Snap In-Circuit Debugger
  • MPLAB REAL ICE™ In-Circuit Emulator
  • MPLAB ICE 4 In-Circuit Emulator
  • MPLAB PM3 Device Programmer
  • MPLAB SIM Software Simulator
  • MPLAB Starter Kit on Board (PKOB)
  • EDBG (embedded debugger)
  • Licensed third party debuggers

10   Upgrading the Development Tool Operating System

11   Updating the Tool and Device Packs

When the latest version of MPLAB X IDE is installed with the latest device and tool packs, these packs  will also be available in the MDB. No additional action is needed.
