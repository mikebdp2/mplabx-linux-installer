1       Overview

MPLAB Data Visualizer is a free debugging tool that graphically displays run-time variables in an embedded application. For more information, see:

https://www.microchip.com/en-us/development-tools-tools-and-software/embedded-software-center/mplab-data-visualizer

2       Installation

There are two ways to install MPLAB Data Visualizer on your system: in MPLAB X IDE as a plugin or as a stand-alone application.

 

The MPLAB Data Visualizer plugin in now part of MPLAB X IDE as of v5.50.

To access the data visualizer in MPLAB X IDE, select Window>Debugging>Data Visualizer or Window>Debugging>Power Monitoring to open the data visualizer for power monitoring.

To see if an updated plugin version is available, go to Tools>Plugins>Updates and click Check for Updates. For more information on plugins, see the MPLAB X IDE documentation, “Add Plugin Tools.”

Note: If you have problems finding or installing an updated plugin, see the text below for how to download the plugin. Also see MPLAB X IDE documentation for how to install a downloaded plugin.

 

To install as a stand-alone app:

  1. Go to https://gallery.microchip.com/.
  2. Search for “mplab data” to find the standalone app. The plugin is also available here if needed.
  3. Click the link to go to a page of information about the standalone app. Click “Download”.
  4. Once downloaded, double click to install the application.

3       Release Information

To find release notes for the plugin or the stand-alone app:

  1. In MPLAB X IDE or MPLAB Data Visualizer, select Help>Help Contents.
  2. Under “MPLAB Data Visualizer”, select the “About” or “Release Notes” topic.
  3. Click on the “Release Notes” link to open the current release notes.

 

Alternatively, to find release notes for the plugin:

  1. In MPLAB X IDE, select Tools>Plugins.
  2. On the Installed tab, find and click on “MPLAB Data Visualizer”.
  3. View the release notes in the right pane.

4       Help File

To find help for the plugin or the stand-alone app:

  1. In MPLAB X IDE or MPLAB Data Visualizer, select Help>Help Contents.
  2. Under “MPLAB Data Visualizer”, select the “About” or “User Guide” topic.
  3. Click on the “Help Guide” link to open the current WebHelp.
