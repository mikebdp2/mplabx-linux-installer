Procedure

SCL offers limited code reusability via the procedure construct. Procedures can be inserted in processes, and then may be called from within the process body. At this time procedures cannot take parameters.

Here is the basic syntax of a procedure. 

process is
   procedure procName() is
       begin
       //procedure statements go here
       end procedure procName;
   begin                //begin process body
   procName();          //call to procedure
   end process;

Here is a more concrete example: 

process is
   procedure HitSwitch() is
       begin
       RD1 <= '1';\n    //user button on pin RD1
       end procedure procName;
   begin
   HitSwitch();         //simulate user hitting the switch
   end process;
