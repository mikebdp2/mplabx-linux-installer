PKB		 Tuples (ANDed BPs) Maximum number of Tuples (ANDed breakpoints). A tuple consists of one program memory breakpoint ANDed with one data memory breakpoint. Breakpoint 1 AND breakpoint 2 must occur at the same time before a program halt.
Sequenced BPs Maximum number of sequences and sequenced breakpoints.
XS/YBPS: X is the maximum number of a breakpoint sequences (S). Y is the maximum number of breakpoints (BPS) in a sequence.
A breakpoint sequence is a list of breakpoints that execute but do not halt until the last breakpoint is executed.

Table R/W BPs Break on a table read or write.
