Simulator Debug Features Table - Heading Descriptions

Feature Feature Description*
Device Device has debug support in the current version of MPLAB® X IDE.
Prog/Data BPs Maximum number of Program and Data Memory breakpoints.
X Total: X breakpoints are shared between program and data memory.
Delay Count Maximum number of delay counts.
Break after a delay occurs specified by a counter.
Pass Count Maximum number of pass counts.
Break after a number of passes occur specified by a counter.
Data BP Access Break on data fetch, data write or either data fetch or write.
Tuples (ANDed BPs) Maximum number of Tuples (ANDed breakpoints).
A tuple consists of one program memory breakpoint ANDed with one data memory breakpoint.
Breakpoint 1 AND breakpoint 2 must occur at the same time before a program halt.
Sequenced BPs Maximum number of sequences and sequenced breakpoints.
X/Y: X is the maximum number of a breakpoint sequences. Y is the maximum number of breakpoints in the sequence.
A breakpoint sequence is a list of breakpoints that execute but do not halt until the last breakpoint is executed.
Stopwatch Use the stopwatch with breakpoints to time code execution. Stopwatch uses 1-2 breakpoint resources
Table R/W BPs Break on a table read or write.
Reset Break Break on reset.
Sleep Wake Break Break on wake from Sleep.
Stack Over/Under Break Break on stack overflow/underflow.
WDT Timeout Break Break on Watchdog Timer timeout.
Runtime Watch Display changing variable values in a Watches window.
Trace Simulator Trace
Allows you to record the step-by-step execution of your code and examine this recording.
Simulation speed is reduced by up to 50% when using trace.
Analyzer Simulator Logic Analyzer
Allows you to graphically view digital signals over a defined time period.
Register Trace Simulator Register Trace
Allows you to record changes in Register values and Global Variable values over a defined time period.
* See MPLAB X Simulator Help
