1       Overview

MPLAB® Analysis Tool Suite is a collection of analysis tools integrated into the MPLAB X Integrated Development Environment (IDE). It supports all Microchip MCU, MPU and CEC devices and offers a code coverage feature and a Motor Industry Software Reliability Association (MISRA®) check in the IDE.

The MPLAB Analysis Tool Suite requires Microsoft Windows® 10 or later, macOS® or the latest versions of Linux® (Ubuntu, Debian, Centos) operating systems in addition to the following:

  • MPLAB X IDE v6.00 or greater.
  • An MPLAB Analysis Tool Suite license for workstation, server and HPA. Licenses can be purchased on Microchip Direct (https://www.microchipdirect.com/).
  • An MPLAB XC C compiler version, either FREE or PRO, that supports the MPLAB Analysis Tool Suite: MPLAB XC8 v2.35, MPLAB XC16 v2.0, or MPLAB XC32 v4.0.

2       Code Coverage

The code coverage feature provides visibility to the parts of your code that have been executed.

Code Coverage Features:

  • Fully Integrated into MPLAB X IDE

-       Source code lines highlighted to show which lines executed or not

-       Navigation tools that quickly move between files and functions

  • Configurable Summary View

-       Dashboards that quickly indicate key metrics

-       Sortable, collapsible list of statement and instruction coverage

-       Ability to see coverage percentage and number of executed and total lines

  • Custom Reports

-       Ability to output source files view and summary view information

-       User-selectable content

-       Multiple output formats (HTML or CSV)

Code Coverage on the Command Line:

When running the compilers with the MATS license on the command line, code coverage output is now available as of MPLAB X IDE v6.05.

Please see the “MPLAB Analysis Tool Suite User’s Guide” (DS-50003236), 2.9 “Command Line Support for Code Coverage in MDB”.

3       MISRA Check

The MISRA check in the IDE provides guidelines and static code checking to ensure safe, secure, portable and reliable C/C++ code in embedded systems.

MISRA Check Features:

  • Fully Integrated into MPLAB X IDE

-       Run static code analysis against MISRA C:2012 rule set to ensure high code quality

-       Execute MISRA checker from the MPLAB X IDE menus (as part of a post-build step) or from the command line

-       MISRA check a single file or entire project

-       View violations to the MISRA rules in output view and navigate quickly and easily to the affected source code location

  • Configurable MISRA Check Tab View

-       Option to select or deselect application of specific MISRA rules for all files

-       Ability to add comments in code to disable a specific rule to a specific line of code

-       Ability to save or restore selection of applied rule sets

  • Custom Reports

-       Documentation of quality by creating reports

-       Reports containing summary of rule violations with list of rules applied and checked

-       Ability to output reports in HTML or CSV

4       Thread Safety Check

The Thread Safety Check locates static local objects used by multiple threads.

Thread Safety Check Features:

  • Fully Integrated into MPLAB X IDE

-       Execute Thread Safety checker from the MPLAB X IDE menus or from the command line

-       Thread Safety check a single file

-       View violations in output view

 

5       What’s New in this Release

  • Exclude files from MISRA analysis on the MISRA Check tab (Tools>Options>Embedded) under “Additional Cppcheck arguments” or on the command line using --cppcheckArgs (use –-help for args) or –-excludePattern.
  • Thread Safety Check added.
  • Option to select source lines (Code) and output in HTML format with Code Coverage information.
